## 2. Telnet

Telnet 접속

```bash
-yum install telnet-server.x86_64
-systemctl start telnet.socket
-systemctl status telnet.socket
-systemctl enable telnet.socket
-firewall-cmd --permanent --add-port=23/tcp
firewall-cmd --reload

Centos>ifconfig 
ubuntu>telnet CentosIp
```
